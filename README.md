## 📚 学习笔记

### 📌 命名规范（笔记文件）

为保证笔记文件清晰易维护，采用统一命名方式：

```log
cpp-[分类]-[关键词].md
```

说明：

- `[分类]`：使用小写英文，反映技术领域，如 `stl`、`algo`、`cmake`
- `[关键词]`：使用驼峰风格拼写重点内容，如 `VectorErase`、`BinarySearch`
